About

Built with curiosity, code, and care.

I'm Anika, a student developer who enjoys building websites that turn ideas into interactive digital experiences. Anika Builds is my dedicated space to organize the websites I have coded, designed, improved, and published.

This archive focuses only on my website projects, from AI tools and climate dashboards to creative experiments and personal web experiences. Each project helped me practice layout design, responsive interfaces, JavaScript interactions, APIs, user experience, and the full process of turning an idea into a live link.

My goal with Anika Builds is to keep a clean, client-friendly record of my web development work as it grows. I will continue adding new launched websites here, updating project details, and using this space as a polished hub for everything I build on the web.
